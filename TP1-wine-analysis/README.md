# TP1 - Analyse du Dataset Wine

## Objectifs
- Analyse exploratoire complète du dataset Wine de l'UCI
- Utilisation de matplotlib et seaborn pour la visualisation
- Interprétation des graphiques et tendances

## Structure
- `data/` : Contient les fichiers wine.data et wine.names
- `notebooks/` : Notebooks Jupyter d'analyse
- `figures/` : Graphiques générés
- `results/` : Résultats et rapports

## Dataset
**Source:** UCI Machine Learning Repository  
**Lien:** https://archive.ics.uci.edu/ml/datasets/Wine  
**Description:** Chemical analysis of wines grown in the same region in Italy but derived from three different cultivars.

## Étapes d'analyse
1. Chargement et nettoyage des données
2. Statistiques descriptives
3. Visualisations univariées
4. Visualisations multivariées
5. Analyse des corrélations
6. Interprétation

## Technologies
- Python 3
- Pandas, NumPy
- Matplotlib, Seaborn
- Jupyter Notebook
